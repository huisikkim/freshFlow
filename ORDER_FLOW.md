# 주문 흐름 가이드

## 전체 주문 프로세스

```
1. 상품 탐색
   ↓
2. 상품 상세 보기
   ↓
3. 장바구니 담기 (수량 선택)
   ↓
4. 장바구니 페이지 (수량 조절, 삭제)
   ↓
5. 주문하기 버튼 클릭
   ↓
6. 주문 확인 페이지 (배송 정보 입력)
   ↓
7. 주문 완료
```

## 각 단계별 UI

### 1. 상품 상세 페이지 (ProductDetailPage)
- **위치**: `lib/presentation/pages/product_detail_page.dart`
- **기능**:
  - 상품 정보 표시 (이미지, 이름, 가격, 재고, 배송 정보)
  - 하단에 "장바구니 담기" 버튼
  - 수량 선택 다이얼로그

### 2. 장바구니 페이지 (CartPage)
- **위치**: `lib/presentation/pages/cart_page.dart`
- **기능**:
  - 담긴 상품 목록 표시
  - 수량 증가/감소 (+/- 버튼)
  - 개별 상품 삭제 (X 버튼)
  - 전체 삭제 (상단 "전체 삭제" 버튼)
  - 총 상품 수 및 총 금액 표시
  - **하단에 "주문하기" 버튼** ⭐

### 3. 주문 확인 페이지 (OrderConfirmationPage) ⭐ NEW
- **위치**: `lib/presentation/pages/order_confirmation_page.dart`
- **기능**:
  - 유통업체 정보 표시
  - 주문 상품 목록 확인
  - 배송 정보 입력:
    - 배송 주소 (필수)
    - 희망 배송일 (날짜 선택)
    - 배송 요청사항 (선택)
  - 결제 금액 표시 (상품 금액 + 배송비)
  - **하단에 "주문하기" 버튼**

### 4. 주문 완료 다이얼로그
- 주문 성공 메시지
- 홈으로 돌아가기 버튼

## 사용 방법

### 가게 사장님 (STORE_OWNER)

1. **상품 찾기**
   ```
   홈 → 유통업체 찾기 → 유통업체 선택 → 상품 목록
   ```

2. **장바구니에 담기**
   ```
   상품 클릭 → 상세 페이지 → "장바구니 담기" 버튼
   → 수량 선택 → "담기" 버튼
   ```

3. **장바구니 확인**
   ```
   장바구니 페이지로 이동
   → 수량 조절 (필요시)
   → 불필요한 상품 삭제 (필요시)
   ```

4. **주문하기**
   ```
   장바구니 페이지 하단 → "주문하기" 버튼 클릭
   → 주문 확인 페이지로 이동
   ```

5. **배송 정보 입력**
   ```
   배송 주소 입력 (필수)
   희망 배송일 선택
   배송 요청사항 입력 (선택)
   ```

6. **최종 주문**
   ```
   하단 "주문하기" 버튼 클릭
   → 확인 다이얼로그에서 "주문하기" 클릭
   → 주문 완료!
   ```

## 주요 버튼 위치

### 상품 상세 페이지
- **장바구니 담기**: 화면 하단 고정 버튼 (초록색)

### 장바구니 페이지
- **전체 삭제**: 상단 우측 (빨간색 텍스트)
- **개별 삭제**: 각 상품 우측 X 버튼
- **수량 조절**: 각 상품의 +/- 버튼
- **주문하기**: 화면 하단 고정 버튼 (초록색) ⭐

### 주문 확인 페이지
- **주문하기**: 화면 하단 고정 버튼 (초록색, 금액 표시) ⭐
