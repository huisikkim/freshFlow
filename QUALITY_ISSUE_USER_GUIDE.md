# 품질 이슈 신고 사용 가이드

## 📱 가게사장님 사용 방법

### 1. 주문 상세 화면에서 신고하기

배송이 완료된 주문에서 상품에 문제가 있을 때:

1. **주문 목록** → **주문 상세** 이동
2. 배송 완료 상태에서 **"품질 문제가 있나요?"** 섹션 확인
3. **"품질 이슈 신고하기"** 버튼 클릭
4. 신고 양식 작성:
   - 주문 정보 (자동 입력됨)
   - 품목 ID 및 품목명 입력
   - 이슈 유형 선택 (품질 불량, 오배송, 파손 등)
   - 사진 URL 추가 (여러 장 가능)
   - 상세 설명 작성
   - 환불/교환 선택
5. **"신고하기"** 버튼 클릭

### 2. 품질 이슈 목록 확인하기

내가 신고한 품질 이슈 확인:

1. 메뉴에서 **"품질 이슈 관리"** 선택
2. 신고한 이슈 목록 확인
3. 각 이슈의 상태 확인:
   - 🟠 **접수됨** - 신고 완료
   - 🔵 **검토 중** - 유통업자가 확인 중
   - 🟢 **승인됨** - 환불/교환 승인
   - 🔴 **거절됨** - 신고 거절
   - 🟣 **수거 예정** - 수거 일정 잡힘
   - 🔷 **수거 완료** - 불량품 수거 완료
   - 🟦 **환불/교환 완료** - 처리 완료

### 3. 이슈 상세 확인

1. 이슈 목록에서 항목 클릭
2. 상세 정보 확인:
   - 신고 내용
   - 사진
   - 유통업자 코멘트
   - 처리 상태
   - 수거 예정 시간
   - 완료 노트

## 🏢 유통업자 사용 방법

### 1. 대기 중인 이슈 확인

1. 메뉴에서 **"대기 중인 품질 이슈"** 선택
2. 신고된 이슈 목록 확인
3. 각 이슈의 사진 및 설명 확인

### 2. 이슈 승인/거절

각 이슈 카드에서:

**승인하기:**
1. **"승인"** 버튼 클릭
2. 코멘트 입력 (예: "확인했습니다. 환불 처리하겠습니다.")
3. 확인

**거절하기:**
1. **"거절"** 버튼 클릭
2. 거절 사유 입력 (예: "사진 확인 결과 정상 제품입니다.")
3. 확인

### 3. 수거 예약 (추후 구현)

승인된 이슈에 대해:
1. 수거 날짜/시간 선택
2. 수거 예약 완료

### 4. 수거 완료 처리 (추후 구현)

1. 실제 수거 후 "수거 완료" 버튼 클릭
2. 수거 완료 처리

### 5. 환불/교환 완료 (추후 구현)

1. 환불 또는 교환 처리 후
2. 완료 노트 입력
3. 완료 처리

## 💡 팁

### 가게사장님

1. **사진을 명확하게 찍으세요**
   - 문제 부분이 잘 보이도록
   - 여러 각도에서 촬영
   - 전체와 세부 모두 촬영

2. **상세하게 설명하세요**
   - 어떤 문제인지
   - 언제 발견했는지
   - 얼마나 심각한지

3. **빠르게 신고하세요**
   - 배송 완료 후 즉시 확인
   - 문제 발견 시 바로 신고

### 유통업자

1. **신속하게 대응하세요**
   - 신고 접수 후 빠른 확인
   - 24시간 내 승인/거절 결정

2. **명확한 코멘트를 남기세요**
   - 승인 사유 설명
   - 거절 시 이유 명시

3. **약속을 지키세요**
   - 수거 예정 시간 준수
   - 환불/교환 신속 처리
