# 공동구매 기능 접근 방법

## 🎯 유통업자 (공동구매 방 만들기)

### 방법 1: 홈 화면에서
1. 앱 실행 후 **홈** 탭 선택
2. 상단에 보이는 **"📦 공동구매 방 관리"** 초록색 배너 클릭
3. **"공동구매 방 관리"** 페이지로 이동
4. 우측 하단 **"방 만들기"** 플로팅 버튼 클릭
5. 정보 입력 후 **"공동구매 방 만들기"** 버튼 클릭

### 방법 2: 더보기 메뉴에서
1. 앱 하단 네비게이션에서 **"더보기"** 탭 선택
2. **"공동구매"** 섹션에서 **"공동구매 방 관리"** 메뉴 클릭
3. **"공동구매 방 관리"** 페이지로 이동
4. 우측 하단 **"방 만들기"** 플로팅 버튼 클릭
5. 정보 입력 후 **"공동구매 방 만들기"** 버튼 클릭

### 방 생성 후 오픈하기
1. 생성된 방은 **"대기중"** 상태
2. 방 카드에서 **"방 오픈하기"** 버튼 클릭
3. 확인 다이얼로그에서 **"오픈"** 클릭
4. 방이 **"오픈"** 상태로 변경
5. 이제 가게들이 참여 가능! 🎉

## 🛒 가게 사장님 (공동구매 참여하기)

### 방법 1: 홈 화면에서
1. 앱 실행 후 **홈** 탭 선택
2. 상단에 보이는 **"🔥 공동구매 특가!"** 초록색 배너 클릭
3. **"공동구매 목록"** 페이지로 이동
4. 원하는 공동구매 방 선택
5. 상세 페이지에서 수량 입력 후 **"참여하기"** 버튼 클릭

### 방법 2: 더보기 메뉴에서
1. 앱 하단 네비게이션에서 **"더보기"** 탭 선택
2. **"공동구매"** 섹션에서 **"공동구매 둘러보기"** 메뉴 클릭
3. **"공동구매 목록"** 페이지로 이동
4. 원하는 공동구매 방 선택
5. 상세 페이지에서 수량 입력 후 **"참여하기"** 버튼 클릭

### 내 참여 내역 확인하기
1. 앱 하단 네비게이션에서 **"더보기"** 탭 선택
2. **"공동구매"** 섹션에서 **"내 참여 내역"** 메뉴 클릭
3. 참여한 공동구매 목록 확인

```

├── [유통업자] 📦 공동구매 방 관리 배너

├── 공동구매 섹션
│   ├── [유통업자] 공동구매 방 관리
│   │   └── DistributorGroupBuyingPage
│   │

│   │   └── GroupBuyingListPage

    └── ...
```

## 🎨 UI 위치

### 홈 화면
- **위치**: 환영 메시지 바로 아래
- **색상**: 초록색 그라데이션 배너
- **아이콘**: 🛍️ 쇼핑백 아이콘
- **텍스트**: 
  - 유통업자: "📦 공동구매 방 관리"
  - 가게: "🔥 공동구매 특가!"

### 더보기 화면
- **위치**: 사용자 정보 카드 아래, 설정 섹션 위
- **섹션명**: "공동구매"
- **메뉴**:
  - 유통업자: "공동구매 방 관리" (초록색 아이콘)
  - 가게: "공동구매 둘러보기" (초록색 아이콘), "내 참여 내역" (파란색 아이콘)

### 공동구매 방 관리 페이지 (유통업자)
- **위치**: 우측 하단
- **버튼**: 플로팅 액션 버튼 (FAB)
- **텍스트**: "방 만들기"
- **아이콘**: ➕ 플러스 아이콘

## 💡 빠른 접근 팁

### 유통업자
1. **가장 빠른 방법**: 홈 화면 → 초록색 배너 클릭 → 방 만들기 버튼
2. **방 관리**: 더보기 → 공동구매 방 관리

### 가게 사장님
1. **가장 빠른 방법**: 홈 화면 → 초록색 배너 클릭 → 방 선택 → 참여
2. **내역 확인**: 더보기 → 내 참여 내역

## 🔍 찾을 수 없을 때

### 배너가 안 보이는 경우
- 앱을 재시작해보세요
- 로그인 상태를 확인하세요
- 사용자 타입(유통업자/가게)을 확인하세요

### 더보기 메뉴가 안 보이는 경우
- 앱 하단 네비게이션 바에서 가장 오른쪽 "더보기" 탭을 선택하세요
- "공동구매" 섹션은 사용자 정보 카드 바로 아래에 있습니다
